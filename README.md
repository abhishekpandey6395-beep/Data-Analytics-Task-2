# Data Analytics Project 2 - Exploratory Data Analysis (EDA)

## 📌 Project Overview
This project focuses on Exploratory Data Analysis (EDA) using an e-commerce dataset. The objective is to understand the data by calculating basic statistics, identifying trends, and finding useful insights.

## 📂 Dataset
The dataset contains order details such as:
- Order ID
- Date
- Customer ID
- Product
- Quantity
- Unit Price
- Payment Method
- Order Status
- Coupon Code
- Total Price

## 🔧 Tools Used
- Microsoft Excel
- Data Cleaning
- Descriptive Statistics
- Pivot Tables
- Charts

## 📊 Analysis Performed
- Calculated Mean, Median, Count, Sum, Min, and Max
- Year-wise Sales Analysis (2023, 2024, 2025)
- Product-wise Analysis
- Order Status Analysis
- Identified Trends and Outliers

## 📈 Key Insights
- Sales were compared across different years.
- Total revenue was analyzed using TotalPrice.
- Customer purchasing patterns were observed.
- Data was summarized using basic statistical measures.

## 🎯 Conclusion
This project demonstrates the use of Excel for performing Exploratory Data Analysis and extracting meaningful business insights from raw data.
